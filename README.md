This repository contains the code for a Point of Sale (POS) system designed for the Beach Side Restaurant. The system allows restaurant employees to take orders, process payments, and generate receipts. It also includes an employee age verification feature to ensure customers are of legal drinking age when ordering alcoholic beverages.

Features:

Interactive menu display and order placement
Age verification for alcohol orders (minimum age: 21)
Integration with an ID Verification Service for additional security
Receipt generation with detailed order summary
